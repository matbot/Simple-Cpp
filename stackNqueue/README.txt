1. Compile the program 'stq' with the included makefile by entering the terminal command:

$make

	inside of the unzipped folder.

2. Run the program with:

$./stq 

3. At the main menu, you may choose whether to work with the stack or queue structure.

4. You may follow onscreen directions for the desired behavior including returning to the
	main menu or exiting the program.
